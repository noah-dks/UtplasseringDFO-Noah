# Utplassering-Golang-2026

Dere må først laste ned GO. Dette gjøres [her](https://go.dev/doc/install).

### Hvordan starte et go prosjekt:

Klone dette repoet og kjør denne kommandoen i mappen.

```bash
go mod init utplassering/dfo
```

## Kalkulator

Dere skal lage en kalkulator som kan gjøre enkle oppgaver som addisjon, subtraksjon, multiplisere og dividere.<br/>
Da brukeren starter programmet skal de kunne putte inn hvilken operasjon de vil gjennomføre.

- "+"
- "-"
- "\*"
- "/"

Derved skal de kunne putte 2 tall inn og få svaret ut.
Da alt er ferdig skal programmet slutte.

Dere trenger ikke å bekymre så mye om feil. Eneste kravet er at programmet kjører og at man kan gjøre de enkleste operasjonene.
