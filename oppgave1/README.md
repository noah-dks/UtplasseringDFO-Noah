# Utplassering-Golang-2026

Hvordan starte et go prosjekt.

Klone dette repoet og kjør denne kommandoen i mappen.
Bytt example/hello-world med navnet på prosjektet. F.eks utplassering/dfo
```bash
go mod init utplassering/dfo
```
## Kalkulator
Dere skal lage en kalkulator som kan gjøre enkle oppgaver som addisjon, subtraksjon, multiplisere og dividere.
Da brukeren starter programmet skal de kunne si hvilken operasjon de vil gjennomføre. 
Derved skal de kunne putte 2 tall inn og få svaret ut.
Da alt er ferdig skal programmet slutte.
